# Document Automation

Supported Documents:
- BOL
- POD
- Load Tender
- Rate Confirmation

Command:
Build a BOL for this customer

System Flow:
1. Open BBL BOL Template
2. Pull customer information
3. Pull shipment information
4. Generate completed BOL
5. Save to Company Folder/BOLs
6. Create Timeline Entry: BOL Generated

Future Support:
- Auto-save to Google Shared Drive
- Auto-attach to Company Profile
- Auto-log activity in Timeline
