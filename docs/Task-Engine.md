# Task Engine

Every task contains:

- Task Name
- Related Company
- Owner
- Due Date
- Priority
- Status
- Created By
- Created Date
- Completed Date

Status:
- Open
- In Progress
- Waiting
- Completed

Priority:
- Critical
- High
- Normal
- Low

Rules:

Every company must have a Next Task.

Next Task = oldest open task for that company.

Task Owner defaults to Opportunity Owner.

Every task must belong to a company.

Automatic Task Creation:

Need Better Number -> Find Better Number

Need New POC -> Find New POC

Call Monday -> Call Contact

Need Email -> Send Email

Quote Request -> Provide Quote

Load Tender -> Review Load Tender

Need Carrier Base -> Build Carrier Base

Build BOL -> Generate BOL

Dormant Prospect -> Re-engage Prospect
