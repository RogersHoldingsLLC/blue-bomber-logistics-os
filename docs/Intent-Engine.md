# Intent Engine

The system should recognize intent, not commands.

Core Intents:
- Contact Problem
- Need New POC
- Follow Up
- Email Request
- Call Request
- Quote Activity
- Project Opportunity
- Customer Conversion
- Equipment Preference
- Routing Preference
- Carrier Development
- Document Request
- Research Request
- Communication Preference
- Win/Loss Information

Flow:
Smart Note -> Intent Detection -> Action
