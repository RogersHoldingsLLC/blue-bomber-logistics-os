# Blue Bomber OS Scope Lock

Build:
- + Prospect
- Tasks
- Attention Required
- Recent Activity
- Company Profile
- Contacts
- Timeline
- Smart Notes
- Current Opportunity
- Next Task

Do Not Build:
- Revenue Dashboards
- KPI Dashboards
- Sales Pipelines
- Forecasting
- Marketing Automation
- Complex Reporting
- Chatbots
- Unnecessary Admin Screens

Decision Rule:
If it increases clicks or admin work, do not build it.
