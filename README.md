# Blue Bomber Logistics OS

Task-first operating system for Blue Bomber Logistics.

Core Principles:
- Tasks drive activity
- Smart Notes drive automation
- Email drives communication
- Simplicity over complexity

Phase 1 Scope:
- Home Screen
- Prospect List
- Company Profile
- Tasks
- Timeline
- Smart Notes

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

Supabase environment values are listed in `.env.example`. Phase 1 includes the Supabase client and database schema, while the UI uses local seed data so the first screen stays fast.
